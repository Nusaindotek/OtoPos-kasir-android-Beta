# OtoPos-kasir-android-Beta
Otopos (otomotif pos) adalah pos kasir amdroid versi Beta.

🚀 Panduan Shortcut (Keyboard)
Gunakan tombol-tombol berikut untuk mempercepat pekerjaan Anda tanpa harus mengklik menu:

Tombol	Fungsi
F2	Aktifkan Mode Servis
F3	Aktifkan Mode Direct Part
F4	Buka Pengaturan (Settings)
F5	Buka Input Part Cepat
Ctrl + Enter	Tambah ke Billing (Keranjang)
Ctrl + P	Proses Checkout & Cetak Struk

🛠️ Catatan Penting
Data: Aplikasi ini menyimpan data di browser Anda (localStorage). Pastikan untuk rutin menggunakan fitur Export Backup di menu Pengaturan > Manajemen Data sebagai cadangan jika sewaktu-waktu cache browser terhapus.

Laporan: Pantau performa bengkel, gaji mekanik, dan pergerakan stok barang melalui tab Laporan di menu Pengaturan.
